Basic Usage

Initialize the carousel with your images and configuration:

const carousel = new ImageCarousel({ container: '.my-carousel', slides: [ { src: 'image1.jpg', alt: 'Description 1' }, { src: 'image2.jpg', alt: 'Description 2' }, { src: 'image3.jpg', alt: 'Description 3' } ], autoplay: true, autoplayDelay: 5000 });

API Methods

carousel.next() - Go to next slide
carousel.previous() - Go to previous slide
carousel.goto(index) - Go to specific slide
carousel.play() - Start autoplay
carousel.pause() - Stop autoplay
carousel.getCurrentIndex() - Get current slide index
carousel.destroy() - Clean up and remove carousel
